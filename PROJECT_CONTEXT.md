# PROJECT_CONTEXT.md

## 项目背景

`wrc-park-vision` 是 WRC 园区管理岗视觉识别项目，同时也是一个 Obsidian vault。项目服务于中国机器人比赛中的“园区管理岗”场景，目标是为机器人或边缘设备提供园区管理相关的视觉识别能力。

当前视觉识别部分关注三类任务：

1. 禁带品检查：识别露营车或入园人员携带物品中的禁带品。
2. 垃圾识别：识别园区场景中的常见垃圾。
3. 不文明行为识别：识别踩踏草坪、吸烟、占用消防通道、站立或躺在长椅上等行为。

## 初步技术方向

- 使用 YOLO11n / YOLO11s 作为实时小模型 baseline。
- 小模型主要负责禁带品、垃圾、人员、长椅等目标检测。
- 对低置信度、遮挡、歧义样本，后续可能使用多模态大模型 / VLM 做兜底复核。
- 当前只记录设计方向，不把模型、数据、接口或硬件环境视为已确认。
