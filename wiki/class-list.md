# Class List

本文件记录初始类别清单。当前清单是需求起点，不代表类别已经冻结。

## 禁带品检查

初始禁带品类别：

- 喷雾罐。
- 滑板。
- 儿童滑板车。
- 烧烤炉 / 炭火炉。
- 音响。
- 卡式炉。
- 轮滑鞋。
- 扩音喇叭。

待确认：

- 是否需要合并或拆分相似类别。
- 是否需要识别“携带者”和“物品”的关系。
- 是否需要将露营车、背包、手提袋等作为辅助类别。

## 垃圾识别

初始垃圾类别：

- 外卖袋。
- 一次性食品盒。
- 塑料饮料瓶。
- 食品塑料包装袋。
- 纸团。
- 空烟盒。

待确认：

- 是否需要更细的垃圾分类。
- 是否需要识别垃圾袋、纸杯、易拉罐等扩展类别。
- 是否只检测垃圾，还是需要判断垃圾是否在不应出现的位置。

## 不文明行为识别

初始行为类别：

- 踩踏草坪。
- 吸烟。
- 占用消防通道。
- 站立在长椅上。
- 躺在长椅上。

待确认：

- 行为是否作为检测类别、规则事件，还是由多模型组合判断。
- 是否需要 pose、tracking、segmentation 或时序判断。
- 是否需要将草坪、消防通道、长椅作为场景目标类别。

## 可能的辅助类别

以下类别可能用于规则判断，但是否纳入第一阶段模型待确认：

- 人员。
- 长椅。
- 草坪区域。
- 消防通道区域。
- 露营车。
- 背包 / 手提袋。
